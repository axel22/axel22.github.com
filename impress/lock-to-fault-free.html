Lock-Freedom

A concurrent program is lock-free if, after some finite number of steps, at least one of the threads completes an operation. 

        
def insert(k: K, v: V, a: Array, d: Int) {
  val i = index(k, d)
  READ(a(i)) match {
    case ch: Array =>
      insert(k, v, ch, d + 1)
    case x: Leaf =>
      val ch = createChild(x, k, v)
      if (!CAS(a(i), x, ch))
        insert(k, v, a, d)
  }
}
