Goal 1

Unify domain-specific control transfer constructs.

Example: iterators 

        val foreach =
          (t: Tree[T], f: T => Unit) =>
            if (t != null) 
              foreach(t.left, f)
              f(t.elem)
              foreach(t.right, f)
            }

Example: Iterators 

        val iterator =
          { (t: Tree[T]) ~>
            if (t != null) 
              foreach(t.left)
              yield(t.elem)
              foreach(t.right)
            }
          }

Example: Async-Await

Example: Async-Await 

      val r: Future[Json] = request()
