The proposed coroutine model

Delimited, stackful, type-safe, first-class coroutines with snapshots.

Implementation using the metaprogramming support.

Delimited 


  val inList = coroutine {
    (b: List[T]) =>
    while (b != Nil) {
      yieldval(b.head)
      b = b.tail
    }
  }

Stackful 


  val inArray = coroutine {
    (t: Array[List[T]]) =>
    var i = 0
    while (i < t.length) {
      inList(t(i))
      i += 1
    }
  }

Type-Safe

inArray is allowed to call inList because they yield the same type. 

val plyList:
  List[T] ~> (T, Unit)
val plyArray:
  Array[List[T]] ~> (T, Unit)
